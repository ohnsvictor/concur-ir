# CONCUR-IR 2026

**Consensus Criteria for Uniform Reporting in Interventional Radiology**

A universal reporting guideline for primary interventional radiology studies, developed through a structured Delphi consensus process. CONCUR-IR covers 12 domains and 93 core items across 5 study designs (Case Series, Cohort, RCT, Registry, Case-Control).

**Live site:** [concur-ir.org](https://concur-ir.org)
